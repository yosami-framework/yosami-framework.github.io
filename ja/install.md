# インストール
Yosamiはnode(9.0.0以上)がインストール済みの環境であれば、簡単に始めることができます。

## パッケージのインストール
任意のディレクトリを作成後、下記のコマンドでパッケージをインストールします。

```shell
$ npm init -y && npm install --save-dev yosami
```

## プロジェクトの初期化
下記コマンドでプロジェクトを初期化します。

```shell
$ ./node_modules/.bin/yosami
```

たったこれだけでインストールは完了です！
